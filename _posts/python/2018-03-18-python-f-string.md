## 正文
f-strings
要使用f-strings，只需在字符串前加上f，语法格式如下：
```
f ' <text> { <expression> <optional !s, !r, or !a> <optional : format specifier> } <text> ... '
```

* 支持表达式
* 排版格式
* 数字操作
